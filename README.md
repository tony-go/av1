# CaptureAV1

## Setup

You need to install:

```shell
brew install ffmpeg
brew install sdl2
```

```shell
bare-make generate
bare-make build
```

## Run

```shell
./build/capture
```

https://github.com/user-attachments/assets/0d75dd62-e2f3-4ed8-9498-31b72d715340

### Using av1 codecs


```shell
./build/capture --av1
```
